
# Civic Problem Reporter MVP

## Features
- Submit civic complaints
- Upload image
- Auto authority routing
- Complaint storage (in-memory)
- Simple frontend + backend

---

## Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend runs on:
http://127.0.0.1:8000

---

## Run Frontend

Open:
frontend/index.html

in your browser.
